Semi-auto procedure to port pages from groups.oist.jp/scs/ to ${gitrepo}/

  1. Change directory to ${gitrepo}
  2. Convert the BASENAME.md to ${gitrepo}/BASENAME.html using a script named 'fixmd.py'
    e.g. ./bin/fixmd.py -i ../org_docs/getting-started/getting-started.md > ./getting-started.html
  3. Copy image files to ${gitrepo}/assets/images/BASENAME/
    4. e.g. cp ../org_docs/getting-started/*.png ./assets/images/getting-started/
  4. Remove the first part (~90 lines) and the last part (~160 lines) by hand.
  5. Add vertical lines (delimiters) to tables. Without those jeykill does not recognize them as tables.
  6. Run make dev to locally view the page at http://127.0.0.1:4000/
  7. Run make deploy to update the production page at https://gyawai.github.io/pages1/
